# GitHub Repository Details

## Repository Description

Browser-based AI face verification system using face-api.js, FastAPI, Supabase pgvector, Render, and Netlify. Generates face vectors in the browser and stores only vector data.

## Important Repo Notes

- Do not upload backend/.env.
- Upload backend/.env.example only.
- Keep Supabase secret key only in Render environment variables and local backend/.env.
- Frontend is deployed from the frontend folder on Netlify.
- Backend is deployed from the backend folder on Render.
